# Brain UI Features

Web UI for Shipwright Brain — browse, search, and read memories stored in Brain's knowledge base.

Built with SvelteKit 2, Svelte 5, Tailwind CSS 4, and shadcn-svelte. Designed with a shipwright/nautical theme, dark-first with light mode support.

## Feature Areas

- [Browse](browse/memory.md) — navigate memories by kind with status/tag filters
- [Memory Detail](memory-detail/memory.md) — rich markdown rendering with lightbox, code highlighting, refs, attachments
- [Search](search/memory.md) — full-text search with tag and status filtering
- [Navigation](navigation/memory.md) — header, breadcrumbs, command palette (Cmd+K), inline search
- [Error Handling](error-handling/memory.md) — themed error pages with animated SVG illustrations
- Board — kanban view with Not Started / In Progress / Done columns, tag filtering
- [Theming & Branding](theming-amp-branding/memory.md) — dark/light/system theme, shipwright illustrations, custom color tokens
